# Instagram-Thrift-Store-Database-Design

The repository contains the Instagram Thrift Store Database Design. The store is a small buisness which recieves its orders through Instagram DMs and WhatsApp. The store sells primary two types of products, Thrift and Handmade.

The entities in the database design are as follows: 

1. Customer
2. Order
3. OrderItems
4. Product
5. Shipment
6. Payment
7. ThriftProduct
8. HandmadeProduct

The functionalities depicted in the database design are:

1. Customers can make multiple orders, and thus, multiple payments.
2. One order can contain multiple products.
3. Products are divided into categories of Thrift Products and Handmade Products.
4. Product description includes its category, size, color, stock, condition, brand and era if thrift, artisan name and material if handmade.
5. Separate section is provided for shipments and payments.

The exported diagram image is included in the root of this repository.
